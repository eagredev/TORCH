# Contributing

TORCH is a personal project in active development. It is currently **source-available, not open source** — the code is public for reference, but redistribution and derivative works are not permitted under the current license.

## Pull requests

PRs are not being accepted at this time. This will change when the project moves to an open source license. If you have something you'd like to contribute, open an issue to discuss it first so the work isn't wasted.

## Bug reports and feature requests

Issues are welcome. There are no guaranteed response times or commitments to resolve specific requests — this is a solo project maintained alongside other work. That said, issues are read and good bug reports (clear reproduction steps, expansion version, OS) are genuinely useful.

## When the license changes

TORCH is expected to move to MIT at a future release. When that happens, this document will be updated with a proper contribution guide.
